Pathfinder Program

Project Description

PathFinder is a desktop application designed to help users manage and navigate different locations. The application is built on a graph data structure, which lets users:

  • Add places (nodes)
  • Add connections (edges) between places
  • Find the best possible route

The whole application follows object-oriented principles, and includes features for saving and loading data for secure network information.

This application was made using Live Share in VS Code in a group of two which increased my teamwork skills. The project gave me practical experience in implementing data structures and designing an easy-to-use JavaFX interface, while working with others.

Technologies Used
