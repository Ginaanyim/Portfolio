Pathfinder Program

GitHub

Project Description

This project was a desktop application built with JavaFX, designed to help users manage and navigate a network of different locations. We built the core of the app around a graph data structure, which lets users add places, set up connections between them, and find the best possible route using algorithms. The whole application follows object-oriented principles, and we also added features for saving and loading data for secure network information.

I worked with two teammates on this project on Live Share in VS Code. This increased my teamwork skills. The project gave me practical experience in implementing data structures, designing an easy-to-use JavaFX interface, while working with others. A specific challenge was visually representing the dynamic graph structure within the JavaFX framework, which required careful integration between back-end logic and the front-end display.
