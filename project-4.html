Prototype for AI friend-App

Figma

Project Description

This project was an assignment to take an existing application and redesign its core purpose, which was a challenge. I developed the concept for "Al-buddy and language teacher", a modified version made to support newly arrived individuals in Sweden with their integration.

The application's core purpose is to provide daily mobile support:

  • Language: To support individualized Swedish language learning.
  • Emotional and Cultural Support: To offer emotional support and increase the understanding of Swedish culture and everyday situations.

I designed this application concept and the user interface using Figma, creating an interactive prototype with wireframes that displays the functionallity of the app. The design is grounded in principles of Calm Technology and Social Computing.
