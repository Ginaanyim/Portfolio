Dog Registry Program

Project Description

This was a Java project where I built a straightforward console-based dog registry system. The primary goal was to demonstrate my ability to work with Object-Oriented Programming (OOP). I designed classes for both Dogs and Owners, ensuring the use of encapsulation to protect data integrity. The most challenging aspect was ensuring the system had a clear structure and that the relationships between the dogs and their owners (e.g., adding, removing, listing) functioned smoothly.

This project served as an excellent way to strengthen my core Java knowledge and writing well-organized code.
