Dog Registry Program

Project Description

The Dog Registry is a console-based Java application developed to manage dog registration and ownership. It focuses on Object-Oriented Programming (OOP) by building a logical system that handles data relationships, as well as principles such as encapsulation, inheritance and classes. The application lets the user:

  • Manage dogs and owners (register or remove objects)
  • Display lists of all dogs and owners
  • Administer ownership by assigning dogs to specific owners or removing association
  • Update dog details, such as increasing their age
  • Navigate the system via an intuitive console menu with clear commands

This project was as a great way to strengthen my Java knowledge and understanding the importance of a clear and logical structure.
