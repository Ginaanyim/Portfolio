Ambessa Coffee Shop

Project Description

Ambessa Coffee Shop is my first fictional e-commerce website. The website is responsive and designed to handle orders placed by customers. It relies on an MySQL databases to manages critical data, including:

  • Product inventory (coffee varieties).
  • Customer data (user registration, sign-ups, and profiles).
  • Orders.

I used PHP to manage form data, user sessions via cookies, and enable secure interactions with the database. The platform also features a direct mail function, allowing customers to send emails straight to the business for support.

I faced challenges with PHP, particularly ensuring secure SQL database interactions and session management, which was a great opportunity to deepen my understanding of server-side logic.
