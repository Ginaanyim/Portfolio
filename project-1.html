Ambessa Coffee Shop

Project Description

Ambessa Coffee Shop is a full-stack e-commerce solution I developed using HTML, CSS, JavaScript and PHP for backend processing. The website is fully responsive and designed to handle complete sales workflows.

This application relies on an SQL database that manages all critical data, including:

  • Product inventory (coffee varieties).
  • Customer data (user registration, sign-ups, and profiles).
  • Order history and processing status.

I used PHP to manage form data, handle user sessions via cookies, and enable secure interactions with the database. The platform also features a direct mail function, allowing customers to send emails straight to the business for inquiries and support.

I tackeled challenges with PHP, particularly ensuring secure SQL database interactions and session management, which was a great opportunity to deepen my understanding of server-side logic.
