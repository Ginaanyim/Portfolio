Quiz Game

Project Description:

An interactive speed-quiz where users must answer a question within 15 seconds. The project focuses on timing logic and real-time user feedback.

  • Timer Logic: Built with JavaScript intervals to handle a 15-second countdown and automatic timeouts.
  • User Interface: Features pop-up instructions and dynamic DOM updates using jQuery to show or hide the game area.
  • Input Handling: Includes validation that ignores case sensitivity and extra spaces for a better user experience.
  • Flow Control: Managed through event listeners that trigger start, reset, and submission functions.

The project demonstrates how to synchronize multiple asynchronous actions, such as stopping a countdown the exact moment a user submits their answer.
